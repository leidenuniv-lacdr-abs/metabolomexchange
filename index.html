October 2012 marked the launch of a global effort to enable free and open sharing of metabolomics data. Coordinated by EMBL-European Bioinformatics Institute, COSMOS (Coordination of Standards in Metabolomics) has brought together European metabolomics data providers to set and promote community standards.

As part of WP5 MetabolomeXchange.org was setup. The initial release was early june 2014 and the formal introduction was during the 2014 Metabolomics conference. MetabolomeXchange.org delivers the mechanisms needed for disseminating the metabolomics data that are being shared within the COSMOS consortium to the metabolomics community at large (both metabolomics researchers and databases). The main objective is to make it easier for metabolomics researchers to become aware of newly released, publicly available, metabolomics datasets that may be useful for their research.

Getting started!

This section is when you want to run your own copy of MetabolomeXchange.org, if you want to become a data provider please contact us at join@metabolomexchange.org.

To run the site you need a webserver capable of running PHP (like Apache 2.x) and MongoDB.

When installed you only need to adjust the /classes/Config.php accordingly. When it detects an empty database it will bootstrap it with the providers.json found in the config, or the default providers.json in the root, and fetch the current publicly available datasets.
