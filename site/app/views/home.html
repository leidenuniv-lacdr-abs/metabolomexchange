Subscribe

MetabolomeXchange uses RSS to allow users to be notified of new or updated datasets. On serveral pages you will see the orange RSS icon that allows you to subscribe to the feed using your own preferred reader. In the top-right corner of this page you will find the link to subscribe to all updates. If you search a smaller icon will appear, just above the search results, that allows you to recieve only updates matching that search term. The last option it to subscribe to only updates from a specific provider, the link to this feed is listed in the overview of the providers.

API

The primairy objective of MetabolomeXchange is to improve and encourage the exchange of Metabolomics data. One of the ways we do this is to provide an API (Application Programmable Interface). Providing a single point of entry when searching for Metabolomics data sets.

Our service architecture has been designed to be light-weight. Therefore the API is limited to act as a thin aggregation layer on top of the actual data providers. The core information aggregation is done server-side, intelligently cached, and exposed by the API. The API is the back-end for this site and the RSS feeds.

The API is versioned, by default you use the latest version available. If you want to use an older (still supported) version you can specify this when calling the API. Further documentation and the version specific information can be found at: http://api.metabolomexchange.org
